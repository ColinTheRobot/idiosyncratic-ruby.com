About Idiosyncratic Ruby

Ruby is not only optimized for developer happiness, but also for writing one-liners and it is still one of the top choices for code golfers. Ruby's grammar definition is more than 10000 lines long and there is no editor or syntax highlighter that actually highlights Ruby correctly. A lot of them already fail at highlighting strings not created with the usual quotes syntax, but with one of the more than 100 other ways. But:

IDIOSYNCRATIC != BAD

What You Will Get

  • Learn how to impress your co-workers with obscure Ruby tricks
  • Learn about lesser-known Ruby command-line options
  • Get a lot of tables comparing things and tell you what to use
  • Master Ruby's less common grammar features

Living Content

Idiosyncratic Ruby is a series of blog posts. All the content is on GitHub, and although all posts will be published in May 2015, the content is meant to live further.

About the Author

Jan Lelis is a freelance developer and he is ruining his Ruby style since late 2009. Back then, he was very unhappy with the IRB command-line experience, so he ended up writing the Irbtools gem, which makes using IRB more enjoyable.

Occasionally, he writes down his Ruby experiences and he micro-blogs on twitter.com, not only about Ruby, but also about JavaScript and full-stack topics, for example, AngularJS and WebRTC.
